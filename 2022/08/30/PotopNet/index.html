  1. The input image is passed into a CNN backbone to get the feature map.
  2. The feature map is seen as several patches with size as 1 x 1 x D.
  3. Each patch is then compared to prototypes to calculate the simlarity between them to obtain the similarty map (activation map).
  4. By max-pooling the similarty map, it will give us the similarity score for each patch.
  5. The similartiy scores are then passed into the fully connected layer to get the final prediction.
